=============
Configuration
=============

You can configure the server with the following fields in ``config.json``

.. code-block:: json

    {
      "port": "SERVEUR_PORT",
      "db": {
        "host": "MONGODB_HOST",
        "port": "MONGODB_PORT",
        "name": "MONGODB_NAME",
      },
      "https": {
        "port": "API_PORT"
      }
    }
